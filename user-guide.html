1. Introduction

The Productivity App is a comprehensive web application designed to help you organize your academic and personal life. It includes the following key features:

  • Notes Management with tagging and categorization
  • Task Management with calendar integration
  • Goal Setting and Progress Tracking
  • Academic Performance Tracking
  • Data Export/Import for portability
  • Dashboard with productivity insights

The app works across all devices (desktop, tablet, and mobile) and stores all data locally in your browser.

2. Getting Started

When you first open the app, you'll see the main dashboard with an overview of your productivity metrics. The navigation menu at the top allows you to switch between different sections of the app.

Navigation

The main navigation menu includes:

  • Dashboard: Overview of all your productivity metrics
  • Notes: Create and manage your notes
  • Tasks: Manage your tasks and to-do lists
  • Calendar: View tasks in a calendar format
  • Goals: Set and track personal goals
  • Marks: Track academic performance
  • Data: Export and import your data

3. Notes Management

Creating a Note

To create a new note:

  1. Navigate to the Notes section
  2. Click the "New Note" button
  3. Enter a title and content for your note
  4. Add tags by typing them in the tags field (separate with commas)
  5. Click "Save Note"

Organizing Notes

You can organize your notes using tags and categories:

  • Tags: Add multiple tags to each note for easy filtering
  • Search: Use the search bar to find notes by title, content, or tags
  • Filter: Click on a tag to filter notes with that tag
  • Archive: Archive notes you no longer need but want to keep for reference

Linking Notes to Tasks

You can link notes to related tasks:

  1. Open a note
  2. Click "Link to Task" button
  3. Select a task from the list or create a new one
  4. The note and task will now be linked, allowing for easy navigation between them

4. Tasks & Calendar

Creating Tasks

To create a new task:

  1. Navigate to the Tasks section
  2. Click the "New Task" button
  3. Enter a title, description, due date, and priority
  4. Assign a category if desired
  5. Click "Save Task"

Managing Subtasks

You can break down complex tasks into subtasks:

  1. Open a task
  2. Scroll to the Subtasks section
  3. Enter a subtask title and click "Add"
  4. Check off subtasks as you complete them

Calendar Views

The Calendar section offers three different views:

  • Month View: Overview of the entire month
  • Week View: Detailed view of the current week
  • Day View: Hour-by-hour view of a specific day

You can create tasks directly from the calendar by clicking on a date or time slot.

5. Goals Management

Setting Goals

To create a new goal:

  1. Navigate to the Goals section
  2. Click the "New Goal" button
  3. Enter a title, description, and target date
  4. Set a category and whether the goal is recurring
  5. Click "Save Goal"

Tracking Progress

You can track your progress toward goals in several ways:

  • Progress Bar: Manually update your progress percentage
  • Milestones: Add specific milestones and check them off as you complete them
  • Streaks: For recurring goals, the app tracks your streak of consecutive days

Completing Goals

When you've achieved a goal:

  1. Open the goal
  2. Click the "Complete" button
  3. The goal will be marked as completed and moved to your completed goals list

6. Marks/Grades Tracking

Adding Subjects

To add a new subject:

  1. Navigate to the Marks section
  2. Click the "New Subject" button
  3. Enter the subject name, term, and instructor
  4. Click "Save Subject"

Adding Grade Components

For each subject, you can add different grade components:

  1. Open a subject
  2. Click "Add Component"
  3. Enter the component name (e.g., "Midterm Exam"), weight percentage, and grade
  4. Click "Save"

Analyzing Performance

The app automatically calculates your final grade based on the weighted components. You can also view a chart showing your grade trend over time for each subject.

7. Data Export/Import

Exporting Data

To export your data:

  1. Navigate to the Data section
  2. Select which data types you want to export (or choose "All Data")
  3. Click "Export Data"
  4. Save the JSON file to your computer

Importing Data

To import previously exported data:

  1. Navigate to the Data section
  2. Click "Choose File" and select your JSON export file
  3. Choose whether to merge with existing data or replace all data
  4. Click "Import Data"
Note: When merging data, items with the same ID will be updated with the imported data. This is useful for transferring data between devices.

8. Dashboard

Overview

The dashboard provides a comprehensive overview of your productivity metrics:

  • Statistics Cards: Quick counts of your tasks, notes, goals, and subjects
  • Task Completion Chart: Visualization of tasks completed over time
  • Goal Progress Chart: Visual representation of your progress on current goals
  • Task Categories Chart: Distribution of tasks across different categories
  • Grade Distribution Chart: Overview of your academic performance
  • Upcoming Tasks: List of tasks due soon
  • Productivity Insights: Personalized insights based on your data

9. Tips & Tricks

Keyboard Shortcuts

Shortcut Action
Ctrl + N Create new item (note, task, goal, etc.)
Ctrl + S Save current item
Ctrl + F Focus on search box
Esc Close modal or cancel current action

Mobile Usage

The app is fully responsive and works on mobile devices. Some tips for mobile usage:

  • Use the bottom navigation bar for quick access to main sections
  • Swipe left/right on calendar to navigate between months
  • Long-press on items for additional options
  • Use the "Export Data" feature regularly to back up your data

Data Privacy

All your data is stored locally in your browser's localStorage. This means:

  • Your data never leaves your device unless you explicitly export it
  • Clearing your browser data will delete your app data
  • Use the export feature regularly to back up your data
  • To transfer data between devices, export from one device and import on the other
